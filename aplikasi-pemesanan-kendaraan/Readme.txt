PANDUAN PENGGUNAAN :
1.KETIK PERINTAH DI TERMINAL : php artisan storage:link
2.UBAH NAMA  DATABASE DI FILE ENV : DB_DATABASE=pemesanan_kendaraan
3.KETIK PERINTAH DI TERMINAL UNTUK MIGRATE : php artisan migrate --seed
4.KETIK PERINTAH DI TERMINAL MENGINSTALL COMMPOSER : composer update / composer install
5.KETIK UNTUK MENJALANKAN APLIKASI : php artisan serve
